# Sleep-Disorder-Data-Analysis-Project
Sleep Disorder Data Analysis project exploring patterns in sleep quality, lifestyle factors, and health indicators to highlight key patterns and support data-driven health insights. It uncovered relationships between sleep duration, stress, BMI, and lifestyle habits. Performed data cleaning, analysis, and visualization to identify trends


## **Objectives:*
Analyze sleep patterns and quality
Identify factors affecting sleep disorders
Explore relationships between stress, BMI, and lifestyle
Generate insights for data-driven health decisions

## **Tools & Technologies:*
SQL 
Python 

## **Key Analysis Performed:*
Data Cleaning & Preprocessing
Handling Missing Values
Exploratory Data Analysis (EDA)
Correlation Analysis (Sleep Duration, Stress, BMI)
Trend Analysis
Data Visualization

## **Dashboard Features:*
Sleep Duration Analysis
Stress Level vs Sleep Quality
BMI Impact on Sleep
Lifestyle Factors (Exercise, Occupation)

## **Dataset Description:*
The dataset includes the following attributes:

Person ID,	Gender,	Age	,Occupation	Sleep Duration,	Quality of Sleep, Physical Activity Level,	Stress Level,	BMI Category	Blood Pressure,	Heart Rate,	Daily Steps,	Sleep Disorder

## **Key Insights:*
Identified key factors impacting sleep quality
Found correlation between stress levels and sleep disorders
Highlighted the role of lifestyle habits in sleep health
Discovered patterns across different BMI categories
